Feedstock for [libdate-tz](https://github.com/HowardHinnant/date) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

A date and time library based on the C++11/14/17 <chrono> header

## Architectures

| Architecture | Latest published |
|--------------|------------------|

| linux-arm64 (`linux-aarch64`) | ✅ `3.0.3` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge libdate-tz
```
